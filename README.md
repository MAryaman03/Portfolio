# Aryaman Mohanty - Developer Portfolio

A premium, futuristic developer portfolio built with modern web technologies.

## Tech Stack
- **React 18**
- **Vite**
- **TypeScript**
- **Tailwind CSS 3**
- **Framer Motion**
- **Lucide React**

## Features
- 🎨 **Glassmorphism Design**: Premium translucent UI elements.
- 🌙 **Dark Mode First**: Optimized for deep purple/blue neon aesthetics.
- ✨ **Framer Motion Animations**: Smooth page transitions, scroll reveals, and micro-interactions.
- 🖱️ **Custom Cursor Glow**: Interactive glow effect following the mouse pointer.
- 📱 **Fully Responsive**: Optimized for desktop, tablet, and mobile.
- ⚡ **Performance Optimized**: Lazy-loaded animations, fast Vite bundling.
- 🔗 **Scroll Spy Navigation**: Active section highlighting.

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```

## Architecture
- `src/components/`: Reusable React components (Section, Hero, About, Projects, etc.)
- `src/lib/`: Utility functions (like Tailwind class merger `cn`).
- `src/index.css`: Global styles, custom scrollbar, typography, and utility classes.
- `tailwind.config.js`: Custom color palette, extended theme, and animations.

## Customization
- **Colors**: Edit `tailwind.config.js` `colors` object to change the primary and secondary accents.
- **Data**: Update the arrays in `Skills.tsx`, `Experience.tsx`, `Projects.tsx`, and `Education.tsx` to modify your content.
- **Typography**: Google Fonts (Inter and Outfit) are configured in `index.html` and `index.css`.
